# Changelog

## 0.1.0

Initial early-access release of PullOps.

- Install a repo-local Workflow Kit for local and GitHub Actions execution.
- Turn Concrete Issues into reviewed, finalized PullOps-Managed PRs.
- Coordinate Spec Issues through dependency-aware Tickets and an Umbrella Branch.
- Run Codex, Claude Code, or another configured agent CLI behind explicit human approval boundaries.
- Record local run state, progress, duration, and supervision evidence.
- Reconcile generated skills, agent docs, labels, and GitHub Actions workflows safely.
